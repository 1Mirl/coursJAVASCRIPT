1- if...

La condition if est l'une des plus utilisées et est également la plus simple à appréhender puisqu'elle va nous permettre d'éxecuter un bloc de code si ET seulement si le résultat vaut true

Attention ! : exceptions

Tout test d'une condition va être évaluée dans contexte booléen; donc à trueà l'exception des valeurs suivantes, qui vont être évaluée à false

  • La valeur 0 (zéro)
  • Le booléen false
  • Une chaine de caractére vide
  • La valeur null
  • La valeur undefined
  • La valeur NaN (Not a Number)

3- Inverser la valeur logique d'un test

Le code placé dans le if n'est exécuté que si le résultat la comparaison est true.

Dans certaines situations, on préfére vérifier que c'est faux en comparant explicitement le résultat de la comparaisosn à false.
