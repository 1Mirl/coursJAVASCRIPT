Introduction

  • Le JavaScript est un langage dynamique
  • JavaScript est un langage principalement côté client
  • JavaScript est un langage interpreté

Le JavaScript est un langage dynamique

Ce langage va nous permettre de manipuler des contenus HTML ou des CSS et de les modifier en fonction de divers évènements ou variables. Un évènement peut être par exemple un clic d'un utilisateur à un certain endroit de la page tandis qu'une variable peut être l'heure de la journée

Le JavaScript est un langage principalement côté client

JS s'éxecute côté client, nous allons quand même pouvoir l'utiliser côté serveur à condition que l'on mette en place un environnement favorable, en utilisant, par exemple node.js

JavaScript est un langage interpreté

Pour être interpreté JS doit être éxécuté directement dans un logiciel intepréteur : tous les navigateurs connus possédent leur intepréteur JS

JavaScript n'est pas JAVA

Encore aujourd'hui, certaines personnes ont tendance à confondre les deux langages "Java" et "JavaScript". Retenez ici que ces deux langages, bien que syntaxiquement assez proches à la base, repose sur des concepts fondamentaux complétement différents et servent à effectuer des tâches totalement différentes
